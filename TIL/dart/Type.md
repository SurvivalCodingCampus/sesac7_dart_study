
### type safe
타입이 항상 일치할 것을 보장

### 타입 추론 vs 타입 명시
```dart
var name = "name"; // 타입 추론
String name = "name"; // 타입 명시
```
타입 추론
- 사용 간편
- 컴파일러가 타입 추론함
- 단, 휴먼 에러 발생 쉬움

타입 명시
- 개발자가 알기 쉬움
- 타입 추론 과정 없음
